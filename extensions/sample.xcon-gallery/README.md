# XCON Operations Dashboard Sample

Opens a production-style XCON/SKETCH dashboard as a Markdown artifact with rendered preview and editable source.

What it shows:

- **Operations Signal Board** as a complete visual surface.
- **Incident Queue** rows with severity, owner, and elapsed time.
- **SLA Risk** and other executive metrics.
- **Live Capacity** signal cards that demonstrate dashboard hierarchy.

Use this sample when an extension needs to generate a visual XCON artifact that looks like a real operational screen while keeping the source available in the same document.
