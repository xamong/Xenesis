# Markdown Operations Brief Sample

Opens an operator-ready Markdown document through `api.openMarkdown()`.

What it shows:

- **Release Readiness Brief** as a useful generated Markdown artifact.
- **Decision Log** for documenting extension-driven operational decisions.
- **Verification Checklist** for repeatable smoke testing.
- `xcon-sketch mode=both` fence content for Markdown + visual artifact handoff.

Use this sample when an extension needs to generate a document that can be saved, reviewed, and shared.
