# Extension Status Console Sample

Opens a production-ready extension panel that demonstrates command registration, dock placement, and scoped runtime metadata.

What it shows:

- **Extension Status Console** as a polished panel surface.
- **Activation Health** for command registration and runtime readiness.
- **Command Surface** for `api.openPanel()` and dock placement.
- **Storage Scope** for extension-local paths and safe display of runtime metadata.

Use this sample when building a first extension panel that should feel like an operator tool rather than a placeholder.
