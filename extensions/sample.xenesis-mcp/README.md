# MCP Operator Runbook Sample

Opens a polished Markdown runbook for local MCP-capable CLI agents.

What it shows:

- **MCP Operator Runbook** as a practical operator document.
- **Capability Registry** discovery before control actions.
- **CR control smoke test** using `xenis_run_extension_command`.
- Safe command and placement patterns for extension-driven Desk actions.

Use this sample when documenting how external agents should inspect and control Xenesis Desk through the bundled MCP server.
