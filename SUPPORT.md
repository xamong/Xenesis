# Support

Use GitHub Discussions for questions, ideas, setup help, and design conversations:

https://github.com/xamong/xenesis-desk/discussions

Use GitHub Issues for reproducible bugs and focused feature requests:

https://github.com/xamong/xenesis-desk/issues

For security-sensitive reports, use the process in [SECURITY.md](SECURITY.md).

When asking for help, include:

- OS and version
- Node.js and npm versions
- Xenesis Desk version or commit
- the command you ran
- the relevant error output
- whether the issue occurs in development, packaged app, or both
